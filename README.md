# GradeMaster Frontend

A minimal web UI for calculating course grades using the GradeMaster API. Enter your midterm, final, weights, and passing grade to see your average and pass/fail status.

## Live Demo

- Application site: https://umutalb.github.io/GradeMaster-Frontend/

## Backend API

The frontend talks to the GradeMaster API:

- GitHub repository: https://github.com/Umutalb/GradeMaster-API

## How to Use

1. Open `index.html` in your browser.
2. Fill in midterm grade, final grade, midterm/final weights, and passing grade.
3. Click **Calculate** to fetch the result from the API and display your average, status, and message.
